# Temperature-Monitoring-System---Bolt-IOT
It is IoT Bolt Wi-Fi module based project with is more useful where temperature should be maintained properly and warn immediately if any problems occured ,especially in pharmaceutical companies.They will receive mail and SMS alert if something went wrong and able to predict future temperatures too.
